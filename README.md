# Lifespace Education - Next.js App

## Quick Setup

1. Copy all these files to your GitHub repo
2. Make sure your GITHUB_TOKEN environment variable is set in Vercel
3. Deploy

## Files Structure
- `app/page.js` - Main page with all content
- `app/api/chat/route.js` - API endpoint for AI chat
- `app/layout.js` - Next.js layout
- `app/globals.css` - All styles
- `package.json` - Dependencies
- `next.config.js` - Next.js configuration
